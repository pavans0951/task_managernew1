# Simple Task Manager

## Description
A simple web application to manage daily tasks using HTML, CSS, and JavaScript.

## Features
- Add a task
- Add task by pressing Enter
- Mark task as completed
- Delete task
- Clean and responsive interface

## Technologies Used
- HTML
- CSS
- JavaScript
